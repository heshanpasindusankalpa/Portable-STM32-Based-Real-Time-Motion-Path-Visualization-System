# Portable-STM32-Based-Real-Time-Motion-Path-Visualization-System2
NaviTrace: A self-contained, real-time motion path visualization system powered by an STM32F407. It leverages an MPU9250 IMU to capture inertial data and renders a live movement trajectory on an SSD1306 OLED via I2C. Built with modular Embedded C, featuring hardware-level optimizations using timers and interrupts.
